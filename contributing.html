The Belgian Puppet Users Group encourages everyone to contribute to this website. We follow just some simple rules :

  • Everything must be puppet related, or related to the Puppet User Community in general
  • Adding content to this site is done by filing a pull request against the Github repos
  • Without prior notification, we only change the category and/or the tag keywords
  • Please file an issue if you find incorrect information, typos etc...
  • Posts should be in the right format

It is not our intention to make another fancy website about puppet, but to be able to share related information.

Information about the system is here Jekyll Introduction

How to create blog post in the markup language is documented here
